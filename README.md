# Power BI Sales Performance Dashboard

This project presents an interactive sales performance dashboard built using Power BI.

## Dataset
The dataset includes:
- OrderID
- Date
- Region
- Salesperson
- Product
- Units
- Revenue
- Month
- Year
- Sales Category

## Dashboard Features
- KPI cards for Total Revenue, Total Units, Total Orders, and Average Revenue
- Interactive slicers for Region, Year, and Sales Category
- Dynamic charts for sales analysis
- Business-focused layout for decision-making

## Tools & Skills
- Power BI
- DAX
- Power Query
- Data Modeling
- Data Visualization
